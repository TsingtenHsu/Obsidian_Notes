# 知识管理中文字类型的分类

知识管理中，文字大概可以分为下面三类：

- 资料
- 信息
- 知识

资料可以是生活中的任何文字，包括从网上剪藏下来的文章和教程。信息则是从资料中找寻出对自己而言有用的文字，去除大部分的修饰。知识则是将信息纳入自己的头脑之中，真正学习到那些信息，可以用自己的话讲述出来。
