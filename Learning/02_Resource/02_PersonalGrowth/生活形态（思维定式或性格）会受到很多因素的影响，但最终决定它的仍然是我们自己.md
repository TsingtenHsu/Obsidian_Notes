# 生活形态（思维定式或性格）会受到很多因素的影响，但最终决定它的仍然是我们自己

[[我们生活在主观世界里]]。这个主观世界，可以被称之为[[思维定式就像地图，我们通过地图去看世界，而非直接感知世界|思维定式]]，也可以被称之为性格。

我们在这个世界上无法控制任何的事情，正如[[生活在这个世上，不要试图掌控他人的生活]]。所以确实在生活之中会有很多元素会影响和控制我们，但要知道的是，即使处在同样的条件之中，所形成的生活形态也是不同的。就如同面临一样的困境，有的人会涅槃重生，有的人会一蹶不振。这种现象足以说明对于生活形态来说，重要的不是我们被赋予了什么，而是我们如何使用被赋予的东西。

固然，生活形态会潜移默化地受到文化、时代、社会背景的影响，但是我们对于这些外在因素所做出的并不会是固定的机械反应，而是可以发挥主观能动性的反应。

[[文化是影响生活形态的重要元素之一|文化确实是影响生活形态的重要元素之一]]，并且对于人们生活形态的影响是潜移默化且悄无声息的，一般处在相同文化背景下的人们会有一些共通的生活形态。

想要改变生活形态，光有决心还不够，首先要做的就是觉知自己真实的社会形态。
