# 情绪稳定的世界是五彩缤纷的

情绪稳定的人不代表无法感知到任何情绪，他们的生理是正常的，并不麻木。相反，这类人能够很好的觉察和接纳自己的真实情绪，能够感受到喜怒哀乐，感受到正面情绪和负面情绪，与之共鸣，并和它们更好的相处。这里也可以引入一个[[情绪稳定的一个特征是弹性情绪|情绪弹性]]的概念。
