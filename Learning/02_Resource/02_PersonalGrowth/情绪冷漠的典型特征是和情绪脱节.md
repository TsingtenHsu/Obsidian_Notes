# 情绪冷漠的典型特征是和情绪脱节

相比于[[情绪稳定的世界是五彩缤纷的|情绪稳定人员能够感知到自己的情绪]]，情绪冷漠人员的典型特征是因为大脑开启了防御机制，潜意识里面抵触体验自己的情绪，试图去逃避、脱离或者回避自身的情绪。

这也是情绪冷漠最大的特征，自我和情绪脱节。
