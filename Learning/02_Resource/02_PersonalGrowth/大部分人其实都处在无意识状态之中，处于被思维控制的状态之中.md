# 大部分人其实都处在无意识状态之中，处于被思维控制的状态之中

无意识状态可以区分为普通的和深层的。[[普通的无意识状态是无限认同于自己的思维、情绪、反应、欲望、好恶]]，[[深层的无意识状态是普通无意识状态的加剧程度]]。

这种无意识状态其实就是我们受自己的思维所控制，认为自己只是思维，并全然地在愤怒和痛苦的情况下去逃避问题，逃避当下。试图从过去去寻找令自己更满意的“当下”。
