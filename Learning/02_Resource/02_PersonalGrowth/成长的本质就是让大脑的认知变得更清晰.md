# 成长的本质就是让大脑的认知变得更清晰

[[我们生活在主观世界里]]，都是通过自己的认知去看待这个世界。

因此，人和人之间，最根本的差距，就是认知能力上的差异。面对同一件事情，不同的人会有不同的看法，这些看法影响判断，而判断，影响我们在每一个当下的选择。我们每时每刻都在做出选择，为自己的命运负责，就是为自己的选择负责。[[我们需要对自己的选择负责]]。

成长的本质就是从根源下手，让大脑的认知变得更加清晰，从而更好地指导我们的现实生活。
