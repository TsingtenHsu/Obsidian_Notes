# 自我成长和改变的前提是自我接纳

[[自我接纳是一种心理状态，是承认自己的一切，不加以评判]]，而自我成长和改变的前提正是自我接纳。

我们想要发生任何改变的时候，肯定都是对当下的自己有所不满。就拿耐心这件事情来说，如果我们想要培养自己的耐心，首先要做的就是承认和接纳当下自己缺乏耐心这一事实开始做起。我们只有接纳本然的自己，才能够寻求改变。
