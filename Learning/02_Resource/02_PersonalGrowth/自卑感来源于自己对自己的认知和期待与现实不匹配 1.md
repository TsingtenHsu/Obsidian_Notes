# 自卑感来源于自己对自己的认知和期待与现实不匹配

[[人们希望摆脱初始的无力状态就是追求优越性 1]]，在追求优越性的过程中，会发现现实中的自己和理想中的自己不匹配，也就是自己的认知和期待与现实不匹配，这样就在心底造成了一种自卑感，但与此同时，我们也会朝着自己所期待的方向去努力和奋斗。
