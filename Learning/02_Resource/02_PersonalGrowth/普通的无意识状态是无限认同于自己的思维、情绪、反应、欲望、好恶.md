# 普通的无意识状态是无限认同于自己的思维、情绪、反应、欲望、好恶

一般来说，人们正常情况下都是处于这种无意识状态，认同思维、情绪、反应、欲望、好恶是自己的全部。

这种状态下，脑袋里会像一种分贝很低的声音一样，一直在作响，如果能够体会到它停下来的状态，才会知道原来自己一直处于无意识认同之中。

在普通的无意识状态下，对[[痛苦的来源是对现实的不接纳和不允许|本然、自我和当下的无限制抗拒]]就造成了我们认为是正常的不安和不满情绪。
