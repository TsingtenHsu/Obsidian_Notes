# 因为耗能，人类天生不喜欢思考

在这个世界上，万物都是为生存而活着。生存需要能量，会引导我们去做低耗能的事情，而思考会消耗大量的能量，因此人类本能地不想思考。

不思考就说明我们受潜意识的控制而去做出各种行动，因此就会陷入到及时行乐和享受之中。所以本能通常都是阻碍学习和成长的，而不学习和成长，又无法做出改变。
