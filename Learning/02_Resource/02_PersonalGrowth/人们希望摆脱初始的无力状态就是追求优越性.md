# 人们希望摆脱初始的无力状态就是追求优越性

人是作为一种无力的存在生活在这个世界上，并且会在后天希望能够通过外在的各种条件去摆脱自身的无力状态，这种无力状态就是追求优越性。也可以理解为追求进步、圆满。但是这种状态作为人们的一种欲望而存在。与之相对应的就是没有达到目标的自卑感，[[自卑感来源于自己对自己的认知和期待与现实不匹配]]。
