# 学到的东西需要即刻运用到自己的身上

我们通过阅读书籍和文章获取了一些知识之后，能够证明我们真正学到的表现就是即刻将其运用到自己的身上。

只有这样，才算是知行合一。才算是满足[[价值观需要行动来捍卫]]。只有做到的，才是我们真正学到的。
