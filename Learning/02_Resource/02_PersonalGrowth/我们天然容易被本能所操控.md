# 我们天然容易被本能所操控

[[因为耗能，人类天生不喜欢思考]]，而不思考不反思带来的结果就是我们在依靠趋易避难的本能在做事情，但是又不知道自己在做什么。

大多数人每天都是被操控，而非主动选择。比如说早上醒来第一眼就是想看手机，过了很长时间之后才意识到自己还在被窝里，不想动身；再比如晚上回顾一天的所作所为，下定决心第二天不能这样，结果一天下来和前一天没有什么变化，陷入到循环之中。
